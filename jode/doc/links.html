JODE Links

Other decompilers

  • A list of decompilers can be found at Marc Meurren's list
  • A very fast decompiler is jad written in C++. It doesn't come with source code though, and misses some features JODE has ;-)
  • A comparison of three decompilers (but not JODE) was done by Dave Dyer.

Other obfuscators

  • Hashjava is another free obfuscator. It is no longer maintained, though, since its successor was commercialized.
  • Zelix Klassmaster does a very good flow optimization and also decrypts strings. But JODE can now undo both.

Graphical User Interface

  • JODE is now used by the JavaInsight plugin for jEdit.
